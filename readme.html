Vinllage

빈(Empty)과 Village(마을)의 합성어인 본 서비스는 AI 기반 카메라 인식 기술을 활용하여 사용자가 스마트폰이나 웹을 통해 실시간으로 쓰레기 분리배출 방법을 안내받을 수 있는 친환경 웹 서비스로, 사진 촬영만으로 분리수거 가능 여부와 방법을 즉시 확인할 수 있으며, 이를 기록·분석하여 개인별 통계와 맞춤형 정보를 제공합니다.

주요 기능

  • 회원의 경우 감지 결과 저장 & 마이페이지 통계 제공
  • 비회원의 경우 최초 이용 후 30일 내 3회 체험 가능
  • 게시판을 통한 정보 공유 & 소통 (회원/비회원 가능)
  • ML Recommendation 결과 시각화

기술 스텍

  • Framework: Spring Boot (Java, Gradle)
  • Database: MySQL, JPA
  • Infra: Docker, AWS(EC2), Nginx
  • Auth: Spring Security, JWT, OAuth2

개발 일정

  • 1주차: 기획 & 설계
  • 2~4주차: 기능 개발
  • 5주차: 테스트, 배포 및 발표

작업 기여도

  • 마이페이지 개발 사용자 정보 관리, 개인정보 수정 및 탈퇴 기능 구현
  • 데이터 시각화 분리수거 이미지 기반 통계 차트 제작
  • 사용자 기록 관리 분리배출 이미지 업로드 및 기록 조회 기능 개발
  • 프로젝트 총괄 디자인 가이드 및 UI/UX 방향성 리드, 백엔드 배포 환경 관리

맹글맹글

전이 학습 기반의 자연어 처리 AI 기술을 활용하여 다양한 영화 속 등장인물과 실제로 대화하는 것과 유사한 경험을 제공하는 채팅 웹 서비스입니다. 사용자는 원하는 캐릭터와 자유롭게 대화를 나눌 수 있습니다.

주요 기능

  • 자연어 처리 AI를 활용하여 채팅을 주고 받을 수 있는 서비스를 제공
  • 전이 학습을 통해 모델마다 특성을 부여하여 대화하는 모델마다 색다름을 제공
  • 대화 내역은 게시판에 기재하여 다른 사람과 공유
  • 댓글을 통해 게시글에 자신의 의사를 표현

기술 스텍

  • Framework: Spring Boot (Java, Gradle)
  • Database: MySQL, JPA
  • Infra: Docker, AWS(EC2), Nginx
  • Auth: Spring Security, JWT, OAuth2

개발 일정

  • 1주차: 기획 & 설계
  • 2~4주차: 기능 개발
  • 5주차: 테스트, 배포 및 발표

작업 기여도

  • 파일 관리 및 공통 작업, 메인 도메인 담당 프로젝트 기본 구조 설계와 공통 기능 구현
  • 템플릿 및 레이아웃 구성 Header, Side, Footer를 포함한 기본 레이아웃과 재사용 가능한 공통 영역 설계
  • 메인 배너 구성 그룹 코드 기반 배너 리스트 조회 및 Thymeleaf 반복 출력 구현
  • Swiper 슬라이드 기능 구현 loop와 autoplay 적용으로 부드러운 슬라이드 구현
  • 백엔드 연동 MainController와 BannerInfoService를 통한 배너 데이터 전달 구현
  • 관리자 배너 관리 기능 구현 배너 등록, 수정, 속성 관리 기능 구현
  • 주요 클래스 및 메서드 개발 배너 조회 및 저장 로직 포함 핵심 클래스 개발
  • 썸네일 출력 유틸 개발 파일 시퀀스와 크기에 맞는 이미지 태그 생성 구현

Quick Draw

Google의 Quick, Draw! 프로젝트를 참고하여 개발한 그림 퀴즈형 웹 서비스입니다. AI 기반 그림 인식 기술을 활용해 사용자가 브라우저에서 직접 그린 낙서를 분석하고, 해당 그림의 정답을 실시간으로 예측하며 다양한 주제의 그림 문제를 바로 제시합니다.

주요 기능

  • 실시간 낙서 인식: 사용자가 그린 그림을 AI가 즉시 분석하여 정답 예측
  • 다양한 카테고리: 동물, 사물 등 여러 주제별 그림 문제 제시
  • 웹 환경 지원: 별도 설치 없이 브라우저에서 실행 가능

기술 스텍

  • Frontend: HTML5, CSS3, JavaScript (Canvas API)
  • Backend: Python (Flask/FastAPI)
  • AI/ML: TensorFlow, Keras, CNN, QuickDraw Dataset
  • Infra: Docker, AWS(EC2, S3)

개선중 개인
Quick Draw

SpringProject

Spring Boot와 Python 기반 머신러닝 모델을 연동하여 위치 기반 식당 추천과 당뇨병 발병 위험 예측을 제공하는 서비스입니다. 사용자의 현재 위치를 기반으로 식당을 추천하고, 당뇨병 고위험군 진단 설문을 통해 개인 맞춤형 건강 정보를 제공합니다.

오늘의 식당

  • 사용자의 현재 위치 기반 식당 추천
  • Kakao 지도 위에 마커 및 인포윈도우 출력
  • Python (KNN) + Spring Boot 연동을 통한 추천 로직
  • 가상 환경 내 머신러닝 모델 실행 및 결과 파싱

당뇨 고위험군 진단 설문 서비스

  • 개인의 생활 습관, 유전자 정보 등 다양한 데이터를 수집 및 분석
  • 머신러닝 기반 당뇨병 발병 위험도 예측 모델 개발

기술 스텍

  • Back-end : Spring Boot, Thymeleaf, Lombok, Spring Data JDBC
  • Front-end : JavaScript, Kakao Map SDK
  • ML Recommendation : Python (Scikit-Learn), KNN, Pickle
  • Infra: Docker, AWS(EC2, S3)
